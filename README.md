# Personal Portfolio Website

This is a portfolio website showcasing skills, services, and projects. It is built using **HTML** and **CSS** and is designed to be simple, responsive, and visually appealing.

## Features

* **Responsive Header Navigation:** Quickly jump to Services, Projects, and Contacts sections.
* **Hero Section:** Introduction with a professional title and social media links.
* **Services Section:** Highlights main offerings:

  * Script Writing
  * Video Production
  * Curriculum Development
* **Projects Section:** Displays selected projects with images, categories, and links to more details:

  * Space Exploration Web Design
  * Birthday Gift Card Website
  * Hangman Game Programmed in C
* **Interactive UI:** Buttons and cards include hover effects for a dynamic experience.

## Tech Stack

* **HTML5** – Structure of the portfolio
* **CSS3** – Styling, layouts, and responsive design
* **Font Awesome** – Social and service icons

## Usage

To run this portfolio locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/hasannader/Your-Repo-Name.git
   ```
2. Open `index.html` in your browser.
3. Make edits to `cv.css` for custom styles or update `index.html` to add new content.
